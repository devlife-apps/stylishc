# :sparkles: stylishc :sparkles:

Create stylish contributors badges with ease!

## Available via...

### :computer: Command Line

#### Install

```shell
git clone <this-repo> stylishc
cd stylishc
npm run build
npm install -g
```

#### Example

```shell
stylishc contributors devlife-apps/stylishc
```
