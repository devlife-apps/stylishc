# stylishc

Create stylish contributors badges with ease!

## Available via...

### Command Line

#### Install

```shell
git clone <this-repo> stylishc
cd stylishc
npm install -g
```

#### Example

```shell
stylishc contributors devlife-apps/stylishc
```
